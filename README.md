# Conti di Casa

App finanze domestiche, mobile-first, PWA installabile, offline-first.

Entrate, uscite, categorie con drag&drop, budget mensili per categoria, dashboard con saldo + torta uscite + trend 12 mesi + barre budget vs reale. Sync Realtime tra dispositivi via Supabase.

**Stack**: HTML + CSS + JavaScript vanilla, Supabase (REST + Realtime), Service Worker, GitHub Pages.

**Deploy**: GitHub Pages servito da `main/root`.

**Backend**: progetto Supabase `lrvkchqvjzynfzevpqaj`, tabelle prefissate `cdc_`.
